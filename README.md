# Hexo 个人博客项目

## 项目概述
基于 Hexo 静态博客框架，使用 AnZhiYu 主题搭建的个人博客。

## 成功验证
- ✅ Hexo 环境配置成功
- ✅ AnZhiYu 主题安装与启用成功
- ✅ 本地服务器 (`hexo server`) 运行正常
- ✅ 访问 `http://localhost:4000` 可查看完整博客界面

## 如何运行
1.  安装依赖：`npm install`
2.  启动本地服务器：`hexo clean && hexo g && hexo s`
3.  在浏览器中访问：`http://localhost:4000`

## 项目结构
主要配置文件：`_config.yml`
主题配置文件：`_config.anzhiyu.yml`
